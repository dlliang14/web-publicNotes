Python 线程池进程池

python-线程池进程池 #

python-并发编程

减少频繁创建和销毁进程/线程的开销，通过复用来提升性能。

进程池 vs 线程池 #

进程池 ProcessPoolExecutor线程池 ThreadPoolExecutor
回调函数执行位置主进程中执行子线程中执行
默认最大并发数CPU 核心数min(32, os.cpu_count() + 4)

核心 API #

from concurrent.futures import ThreadPoolExecutor, ProcessPoolExecutor

pool = ThreadPoolExecutor(max_workers=10)
future = pool.submit(task, arg)          # 提交任务，返回 Future 对象
future.add_done_callback(handle)         # 任务完成后触发回调
future.result()                          # 获取任务返回值（阻塞等待）
pool.shutdown(wait=True)                 # 等待所有任务完成后关闭

示例 #

from concurrent.futures import ThreadPoolExecutor
from threading import current_thread
import time, random

def task(n):
    print(f'{current_thread().name} is running')
    time.sleep(random.randint(1, 3))
    return n ** 2

def handle(future):
    res = future.result()
    print(f"{current_thread().name} 处理结果：{res}")

if __name__ == '__main__':
    # 方式1：手动 shutdown
    pool = ThreadPoolExecutor(max_workers=10)
    for i in range(19):
        pool.submit(task, i).add_done_callback(handle)
    pool.shutdown(wait=True)

    # 方式2：with 自动 shutdown（推荐）
    with ThreadPoolExecutor(max_workers=10) as pool:
        for i in range(19):
            pool.submit(task, i).add_done_callback(handle)
