socketserver 并发 #

socketserver tcp 和 udp 的区别 #

  1. tcp 会每个连接创建一个socket对象，然后每个单独配置一个handle，需要配置循环处理通信
  2. udp 会共享一个socket对象，然后每次接收数据时，都会带上客户端地址信息，发给handle处理

socketserver 源码剖析 #

  • 线程并发 ThreadingMixIn -> ThreadingTCPServer
  • 进程并发 ForkingMixIn (windows 不支持)
  • 通信处理 BaseRequestHandler -> StreamRequestHandler / DatagramRequestHandler
  • serve_forever() -> BaseServer.serve_forever()
    • selector.register(self, selectors.EVENT_READ) 并发，注册事件
    • self._handle_request_noblock() 处理单次请求
      • self.get_request() 获取请求 socket 对象,就是 socket.accept()
      • self.process_request(request, client_address) 处理请求
        • threading.Thread(target=self.process_request_thread, args=(request, client_address)).start() 并发
        • process_request_thread 处理通信
          • self.finish_request(request, client_address) 就是自己定义的handle类

tcpserver 示例 #

import socketserver


class MyHandler(socketserver.BaseRequestHandler):
    def handle(self):
        print("客户端连接：", self.client_address)
        while True:
            try:
                data = self.request.recv(1024)
                if not data:
                    print("客户端断开：", self.client_address)
                    break
                print("收到数据：", data.decode())
                self.request.sendall(data)
            except ConnectionResetError:
                print("客户端强制断开：", self.client_address)
                break
        self.request.close()


if __name__ == "__main__":
    # 参数1 绑定的地址和端口
    # 参数2 要运行的通信循环的代码（带有handle的类）
    # 参数3 bind_and_activate 默认True 立即绑定端口并启动监听
    s = socketserver.ThreadingTCPServer(("localhost", 9999), MyHandler)

    # 循环accept客户端连接，并为每个连接创建一个新的线程来处理通信
    s.serve_forever()

tcpclient 示例 #

import socket


phone = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

phone.connect(('localhost', 9999))
while True:
    msg = input("Enter message to send (or 'exit' to quit): ")
    if msg.lower() == 'exit':
        break
    # if not msg:
    #     continue
    # 默认utf-8编码
    phone.sendall(msg.encode())

    data = phone.recv(1024)
    print(f"Received from server: {data.decode()}")

phone.close()

udpserver 示例 #

import socketserver
import sys

print(sys.version)


class MyHandler(socketserver.BaseRequestHandler):
    def handle(self):
        print("客户端连接：", self.client_address)
        data, sock = self.request
        print("收到数据：", data.decode())
        sock.sendto(data.upper(), self.client_address)


if __name__ == "__main__":
    # 参数1 绑定的地址和端口
    # 参数2 要运行的通信循环的代码（带有handle的类）
    # 参数3 bind_and_activate 默认True 立即绑定端口并启动监听
    s = socketserver.ThreadingUDPServer(("localhost", 9999), MyHandler)
    s.serve_forever()
