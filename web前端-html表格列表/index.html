表格 <table> #

  • <tr> 表格行，<th> 表头单元格，<td> 数据单元格
  • border-collapse: collapse 合并相邻边框
  • rowspan 单元格纵向跨行，colspan 单元格横向跨列
<table style="border: 3px solid #5e9e26; border-collapse: collapse;">
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
    <th>Header 3</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 3</td>
    <td rowspan="2">跨2行</td>
  </tr>
  <tr>
    <td colspan="2">跨2列</td>
  </tr>
</table>

列表 #

标签类型说明
<ul>无序列表list-style-type 控制项目符号样式
<ol>有序列表list-style-type 控制数字/字母样式
<dl>定义列表<dt> 定义术语，<dd> 定义描述
<!-- 无序列表 -->
<ul style="list-style-type: square;">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<!-- 有序列表 -->
<ol style="list-style-type: upper-alpha;">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

<!-- 定义列表 -->
<dl>
  <dt>Term 1</dt>
  <dd>Description 1</dd>
  <dt>Term 2</dt>
  <dd>Description 2</dd>
</dl>
