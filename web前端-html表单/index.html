form 表单 #

multipart/form-data（适合文件上传） #

请求体为分段数据，每段包含字段名和内容。

<form action="/submit" method="post" enctype="multipart/form-data">
  <input type="text" name="name" value="11" />
  <input type="email" name="email" />
  <input type="submit" value="Submit" />
</form>

<!-- 请求体示例：
------WebKitFormBoundaryXxx
Content-Disposition: form-data; name="name"

11
------WebKitFormBoundaryXxx
Content-Disposition: form-data; name="email"

asdad@11
------WebKitFormBoundaryXxx-- -->

application/x-www-form-urlencoded（默认，适合文本） #

请求体格式：key=value&key=value

<form action="/submit" method="post">
  <input type="text" name="name" value="11" />
  <input type="email" name="email" />
  <input type="submit" value="Submit" />
</form>

<!-- 请求体：name=11&email= -->

GET 方式 #

a 标签 #

<a href="/submit?name=11&email=">Submit via Link</a>

JavaScript 跳转 #

<button onclick="submitData()">Submit via JavaScript</button>
<script>
  function submitData() {
    const name = "11";
    const email = "";
    const url = `/submit?name=${encodeURIComponent(name)}&email=${encodeURIComponent(email)}`;
    window.location.href = url;
  }
</script>

AJAX 提交（POST） #

<button onclick="ajaxSubmit()">Submit via AJAX</button>
<script>
  function ajaxSubmit() {
    const xhr = new XMLHttpRequest();
    const params = "name=" + encodeURIComponent("11") + "&email=" + encodeURIComponent("");
    xhr.open("POST", "/submit", true);
    xhr.setRequestHeader("Content-Type", "application/x-www-form-urlencoded");
    xhr.onreadystatechange = function () {
      if (xhr.readyState === 4 && xhr.status === 200) {
        alert("Response: " + xhr.responseText);
      }
    };
    xhr.send(params);
  }
</script>
